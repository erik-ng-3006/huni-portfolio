# Introducing My K-Drama App: Your Personal K-Drama Companion

## The Ultimate Tool for K-Drama Enthusiasts

As a passionate K-Drama fan, I'm excited to share a project I've been working on: [My K-Drama App](https://my-k-dramas.netlify.app/). This application is designed to be your personal companion in the vast world of Korean dramas, helping you discover, track, and manage your watching experience.

## What is My K-Drama App?

My K-Drama App is an open-source web application built with modern technologies to provide K-Drama enthusiasts with a comprehensive tool for exploring and managing their Korean drama journey. Whether you're a seasoned K-Drama veteran or just starting your adventure into this captivating world of storytelling, this app offers features to enhance your viewing experience.

## Key Features

### Discover New Dramas

Browse through an extensive collection of K-Dramas with detailed information about each show, including synopsis, cast, ratings, and more. Our search and filter system makes it easy to find your next binge-worthy show.

### Personal Watchlist

Create and manage your personalized watchlist. Mark dramas as "Plan to Watch," "Currently Watching," "Completed," or "Dropped." Never lose track of where you left off again!

### Track Your Progress

Keep detailed records of your watching progress, including which episode you're on, when you started watching, and when you finished.

### Rate and Review

Share your thoughts by rating and reviewing dramas you've watched. Your insights could help other fans discover their next favorite show.

## Technologies Used

This application is built using a modern tech stack:

-   **Frontend**: React, TypeScript, Material-UI
-   **State Management**: Redux
-   **API Integration**: RESTful API with Axios
-   **Routing**: React Router

## Getting Started with the Project

If you're interested in contributing to the project or just want to try it out locally, follow these steps:

```bash
# Clone the repository
git clone https://github.com/erik-ng-3006/my-kdrama-app.git

# Navigate to the project directory
cd my-kdrama-app

# Install dependencies
npm install

# Start the development server
npm start
```
